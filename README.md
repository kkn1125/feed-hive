# FeedHive MVP 개발

피드하이브는 Go를 기반으로 한 서버사이드 구축, MSA 구현을 목표로 하며, Figma를 이용한 UI/UX 디자인, 화면 설계 후 React 또는 NextJS를 이용한 클라이언트를 개발한다.

- [x] Golang
- [x] Gin
- [x] MariaDB
- [x] RabbitMQ
- [ ] React
- [ ] NextJS
- [ ] Figma
